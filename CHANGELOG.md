# LitFlow 更新日志

## [v2.4.0] - 2026-05-19

### 新增
- 文件解析工具 `lib/file-parser.ts`，支持 PDF / DOCX / Markdown / TXT / HTML 统一提取文本
- 新增 `/api/files/parse` 接口，可接收上传文件并返回解析后的纯文本结果
- 文件类型检测、文件大小校验（单文件默认 20MB）

### 优化
- 为后续“文本超我”“多格式导出”“文件资产管理”打通基础解析链路
- 上传系统从单纯 UI 升级为具备实际解析能力的文件处理模块

---

## [v2.3.0] - 2026-05-19

### 新增
- 可复用拖拽上传组件 `FileDropZone`，基于 react-dropzone 实现
- 支持 PDF / Word / Markdown / TXT / HTML 文件拖拽或点击上传
- 支持文件数量限制、单文件大小限制、上传文件列表展示和删除操作

### 优化
- 新建笔记页和编辑笔记页统一接入可复用上传组件
- 移除页面内重复的上传区逻辑，后续文件解析与导出模块可以直接复用

---

## [v2.2.1] - 2026-05-19

### 修复
- 笔记编辑器保存逻辑改为输出纯文本，避免将 HTML 标签直接写入数据库
- 新建/编辑笔记页字数统计与富文本编辑器联动，显示真实输入字数
- 补充 Tiptap 编辑器样式与 typography 插件配置，确保线上构建稳定通过

---

## [v2.2.0] - 2026-05-19

### 新增
- 笔记富文本编辑器（Tiptap），支持标题、粗体、斜体、下划线、列表、引用、代码块、链接等格式
- 笔记编辑页（`/projects/[id]/notes/[noteId]/edit`），可对已有笔记进行二次修改
- 拖拽附件上传 UI（多文件、大小显示、删除操作）

### 优化
- 新建笔记页从纯文本输入升级为富文本编辑体验
- 编辑笔记页统一使用结构化输入和富文本编辑器
- 编辑器支持 Markdown 风格的快捷输入，同时保留直观工具栏

---

## [v2.1.0] - 2026-05-19

### 新增
- 论文关系知识图谱（D3.js 力导向图可视化）
- 支持/矛盾/延伸/补充四种关系类型，不同颜色和线型区分
- 可拖拽节点、悬停查看关系详情
- 图例标识、关系统计、可展开的关系列表
- 独立知识图谱页面（`/knowledge-graph`），支持选择不同提纲
- 侧边栏"知识图谱"快捷导航入口
- 综述提纲页"查看知识图谱"快速跳转链接

### 优化
- 知识图谱页面空状态引导
- 提纲选择器以卡片形式展示

---

## [v2.0.0] - 2026-05-19

### 新增
- 全新 UI 设计系统，采用春日暖色调（暖杏色 + 春芽绿）
- 优雅衬线字体 Playfair Display（英文标题）+ Noto Serif SC（中文标题）
- 页面进入动画（fadeInUp、staggerIn）和交互动效
- 毛玻璃导航栏和侧边栏效果
- CSS 装饰元素（渐变圆、渐变分割线、空状态几何图形）
- 自定义标签 Pill 样式和笔记类型色条
- 更新日志页面

### 优化
- Landing Page 全面重构，增加数据统计区和可折叠 FAQ
- Dashboard 侧边栏设计，支持手机端滑动菜单
- 整体色彩体系从冷色蓝调改为暖色杏调，更具学术出版物质感
- 卡片和按钮交互反馈更细腻（hover 上浮、active 按压）
